# Sistema de Reservas - FIFO

Proyecto desarrollado en Java que simula un sistema de reservas de vuelos utilizando una estructura de datos FIFO (First In, First Out).

## Funcionalidades
- Reservar asiento
- Cancelar asiento
- Mostrar asientos disponibles

## Tecnologías
- Java
- Programación Orientada a Objetos
- Estructura de datos FIFO
